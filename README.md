**HideMyDiscord** — это простая и эффективная утилита для обхода блокировок Discord, YouTube и Telegram на территории РФ. Работает без VPN, не требует сложных настроек и полностью бесплатна.

Программа является форком популярного инструмента **zapret**, но представляет собой самостоятельный продукт, адаптированный для максимального удобства пользователей.

✨ **Возможности**  
✅ Стабильный доступ к Discord (включая голосовые каналы)  
✅ YouTube без замедлений и ограничений  
✅ Telegram работает как обычно  
✅ Не требуется установка дополнительного ПО  
✅ Мгновенное подключение (запустил .bat и пользуйся)  
✅ Не хранит логи и не собирает данные  
✅ Полностью бесплатно

> ### АНТИВИРУСЫ
> Некоторые антивирусы могут ложно срабатывать на компонент **WinDivert**, который используется программой для фильтрации трафика.  
> WinDivert — это легитимный системный драйвер, подписанный для работы в ядре Windows. Он не является вирусом и не наносит вреда системе.  
> Если ваш антивирус блокирует или помещает в карантин файлы `WinDivert.dll` / `WinDivert64.sys`, добавьте папку с программой в исключения или отключите детектирование потенциально нежелательных приложений (PUA).

## ⚙️ Использование

1. **Включите Secure DNS**  
   - В Chrome: *Настройки → Безопасность → Использовать безопасный DNS* (выберите провайдера, например Google или OpenDNS).  
   - В Firefox: *Настройки → Конфиденциальность и безопасность → DNS через HTTPS* (выберите «Максимальная защита» и укажите, например, `https://dns.google/dns-query`).  
   - В Windows 11: включите DNS-over-HTTPS в настройках сети — [инструкция](https://remontka.pro/dns-over-https-windows-11/).  
   - Если у вас роутер Keenetic, включите опцию «Транзит запросов» в настройках роутера.

2. **Скачайте последнюю версию** с официального сайта [HideMyDiscord](https://hidemydiscord.com) или со страницы релизов.

3. **Разблокируйте архив** (если используется):  
   Нажмите правой кнопкой по скачанному ZIP/RAR → Свойства → внизу поставьте галочку «Разблокировать» (если она есть). Если вы используете 7‑Zip или PeaZip, этот шаг можно пропустить.

4. **Распакуйте** содержимое архива в папку, путь к которой не содержит кириллицу и специальные символы (например, `C:\HideMyDiscord`).

5. **Запустите нужный файл** (см. описание ниже).

## ℹ️ Краткие описания файлов

- **`general.bat`** (и аналоги `general_alt.bat`, `general_fake.bat` и др.) — запуск стратегии обхода **вручную**.  
  Это удобно для тестирования: попробуйте разные стратегии, пока не найдёте ту, которая работает именно у вас. Работоспособность зависит от многих факторов (провайдер, регион, время суток).

- **`service.bat`** — установка в автозапуск и дополнительные функции:  
  - **`Install Service`** — установка выбранной стратегии как службы Windows (автозапуск при старте системы).  
  - **`Remove Services`** — удаление всех служб, связанных с программой.  
  - **`Check Status`** — проверка статуса обхода и установленных служб.  
  - **`Game Filter`** — переключение режима для игр (учитывает UDP и TCP на портах >1023). После переключения требуется перезапуск стратегии.  
  - **`IPSet Filter`** — настройка фильтрации по IP-адресам (полезно при тестировании).  
  - **`Auto-Update Check`** — включение/отключение автоматической проверки обновлений.  
  - **`Update IPSet List`** — обновление списка IP-адресов для обхода.  
  - **`Update Hosts File`** — обновление файла `hosts` (исправляет проблемы с веб-версией Telegram и голосовыми чатами Discord).  
  - **`Check for Updates`** — ручная проверка обновлений.  
  - **`Run Diagnostics`** — диагностика частых проблем (например, конфликты с другими программами или некорректные настройки).  
  - **`Run Tests`** — запуск тестов для проверки работоспособности стратегий на разных сайтах.

## ☑️ Часто задаваемые вопросы и проблемы

### После запуска `general*.bat` ничего не происходит
- Убедитесь, что запуск производится от имени администратора.  
- Проверьте, не блокирует ли антивирус процесс `winws.exe`.  
- Откройте диспетчер задач — процесс должен появиться. Если нет — попробуйте перезапустить ПК и повторить.

### Не работает Telegram (веб-версия) или бесконечное «подключение» в голосовом чате Discord
Запустите `service.bat` → выберите **`Update Hosts File`**. Если файл `hosts` устарел, вам будет предложено обновить его вручную:  
- Скопируйте предложенный текст.  
- Откройте файл `hosts` (обычно лежит в `C:\Windows\System32\drivers\etc\hosts`) блокнотом от имени администратора.  
- Вставьте скопированные строки в конец файла (или замените старые записи). Сохраните и перезапустите программу.

### Обход перестал работать
> **Важно!** Стратегии могут переставать работать из-за изменений в методах блокировки.  
- Проверьте, нет ли ошибок в диагностике (`service.bat → Run Diagnostics`).  
- Убедитесь, что нужные домены или IP присутствуют в списках обхода.  
- Попробуйте другие стратегии (`ALT`, `FAKE` и т.д.).  
- Если ничего не помогает — выполните полную переустановку (см. ниже).

### Как полностью переустановить или обновить?
1. Сохраните свои пользовательские настройки (файлы `*-user.txt`).  
2. Перезагрузите компьютер.  
3. Запустите `service.bat` → **`Remove Services`**.  
4. Удалите папку с программой.  
5. Скачайте свежую версию с сайта, распакуйте в новую папку.  
6. Запускайте `general*.bat` для тестирования, найдите рабочую стратегию.  
7. Для автозапуска: `service.bat` → **`Install Service`** → выберите стратегию.

### Игра или приложение перестали работать при включённом обходе
- Проверьте, что в `service.bat` параметры **`Game Filter`** и **`IPSet Filter`** установлены в `disabled` или `none`.  
- Если проблема остаётся — временно отключите обход (закройте `winws.exe`) и проверьте, работает ли игра без него. Возможно, требуется добавить IP-адреса игры в список исключений.

### Античит ругается на WinDivert
- Прочитайте инструкцию по маскировке WinDivert (адаптированную под HideMyDiscord) — например, используйте специальные утилиты для смены имени драйвера.  
- Либо отключайте обход перед запуском игры.

### Windows 7 требует цифровую подпись драйвера
- Замените файлы `WinDivert.dll` и `WinDivert64.sys` в папке `bin` на версии, предназначенные для Windows 7 (можно взять из соответствующих архивов).
